# EddeBot OBS Plugin Releases

Official releases for the EddeBot Instant Replay OBS Studio plugin.

## Download

Download the latest version from the [Releases](https://github.com/eddedre/eddebot-obs-plugin-releases/releases) page.

## Installation

1. Close OBS Studio if it's running
2. Download the Windows installer (.exe file) from the latest release
3. Run the installer
4. Launch OBS Studio
5. Open **Tools â†’ EddeBot Connection Settings** to configure

## Requirements

- OBS Studio 31.0 or later
- Windows 10/11 x64

## Features

- **Automatic Update Checking**: Plugin checks for updates on OBS startup
- **Instant Replay Management**: Full replay buffer control and automation
- **SignalR Hub Integration**: Real-time communication with EddeBot API
- **Video Duration Detection**: Automatic probing for accurate timing
